## 常见问题 FAQ

- 如何快速判断营养补充剂是否靠谱？
  - 看配方是否公开剂量、是否有第三方检测与原料溯源；对照自身目标（益智补剂/运动营养/复合维生素）做小样本试用并记录体感与训练数据。

- 初次搭建营养方案怎么选起点？
  - 先用“基础模块”打底：优质蛋白或全营养粉 + 复合维生素 + Omega‑3；再按目标加益智/力量/恢复模块，四周复盘一次。

- 如何控制长期成本而不降质？
  - 优先选择配方透明、规格可选的品牌；按周期购买或成套组合，保留2–3个核心单品，避免高重叠成分。
